# TrackVolt — Deploy & Handoff Guide
**Last Updated:** March 5, 2026

---

## Step 1: Verify Build (REQUIRED before anything else)

Open a terminal in `C:\ClaudeWork\Imu\TrackVolt-App\` and run:

```bash
npm run build
```

This runs TypeScript checking + Vite build. **Must pass with ZERO errors.**

If it fails, fix TypeScript errors before proceeding. Common issues:
- Unused imports → remove them
- Missing type imports → add `import type { ... }`

## Step 2: GitHub Push (Recommended Handoff)

### First time setup:
```bash
# Create a private repo on GitHub (via browser or CLI)
# Then from the project folder:

git add .
git commit -m "TrackVolt full source — all 23 pages implemented"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/TrackVolt-App.git
git push -u origin main
```

### After that, for future changes:
```bash
git add .
git commit -m "Description of what changed"
git push
```

### Using GitHub CLI (alternative):
```bash
gh repo create TrackVolt-App --private --source=. --push
```

## Step 3: Deploy to Production (Vercel)

**ONLY do this when you're ready. This overwrites trackvolt.app.**

```bash
# Login (first time only)
npx vercel login

# Deploy
npx vercel --prod
```

The `.vercel/project.json` file already links to the correct Vercel project.

---

## Step 4: Emergency Rollback

If production breaks after a deploy:

1. Go to https://vercel.com/imteyazmdkaish-1195s-projects/crosstrack/deployments
2. Find deployment **GTeJwY1rw** (last known good)
3. Click `...` → **Promote**
4. Hard refresh browser: Ctrl+Shift+R (PWA caches aggressively)

## PWA Install (Phone)

### iPhone:
1. Open trackvolt.app in **Safari**
2. Tap Share button (box with arrow)
3. "Add to Home Screen" → "Add"

### Android:
1. Open trackvolt.app in **Chrome**
2. Tap 3-dot menu → "Install app"
